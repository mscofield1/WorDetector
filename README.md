# WorDetector
Python-based word study and reading distribution

üåì WorDetector: Advanced Reading & Vocabulary Assistant
WorDetector is a comprehensive desktop application designed to streamline the language learning process by combining reading, translation, and vocabulary retention into a single platform. It is specifically crafted for analyzing academic articles and foreign language texts.

üöÄ Key Features
Smart Text Analysis: Seamlessly import PDF, Word (.docx), and TXT files.

Click-to-Translate: Instantly translate selected words into your target language with automatic Part of Speech (POS) analysis (Noun, Verb, Adj, Adv).

Audio-Visual Learning: Real-time Text-to-Speech (TTS) for selected words combined with synchronized text highlighting for better retention.

Dynamic Themes: Fully integrated Dark and Light modes for a comfortable reading experience in any lighting condition.

Vocabulary Marathon (Quiz): Automatically generates multiple-choice quizzes from the words you've studied.

Error Analytics: Provides a detailed report at the end of each quiz, showing incorrect answers alongside their correct translations for effective feedback.
